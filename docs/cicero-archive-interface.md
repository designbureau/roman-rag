# An Interface to the Roman Archive

*System description — DRAFT, to be written.*

A retrieval-augmented reading interface over Cicero's complete works, presented
as a Latin + English parallel text and voiced through a Roman ensemble of
personas. Every answer is grounded strictly in the retrieved passages; the
system is intended to support reading the texts, not to compose new Ciceronian
speeches.
